# CloudBase Relational Database Web Power

A comprehensive guide for accessing CloudBase Relational Database from browser applications using @cloudbase/js-sdk, providing Supabase-style query patterns and best practices.

## Overview

This power provides detailed guidance for integrating CloudBase Relational Database into web applications, including:

- Frontend initialization patterns for CloudBase Relational Database
- Supabase-style query syntax and operations
- Shared database client management across web applications
- React, Vue, and vanilla JavaScript integration examples
- Best practices for browser-based database access

## Key Features

- **Browser Integration**: Seamless CloudBase Relational Database integration for web apps
- **Supabase Compatibility**: Use familiar Supabase-style query patterns
- **Client Management**: Shared database client patterns for optimal performance
- **Framework Support**: Examples for React, Vue, and vanilla JavaScript
- **Query Operations**: Complete CRUD operations with advanced querying
- **Error Handling**: Robust error handling and debugging strategies

## Use Cases

- Building React, Vue, or vanilla JavaScript web applications with CloudBase
- Migrating from Supabase to CloudBase Relational Database
- Creating shared database clients across web application components
- Implementing real-time data operations in browser environments
- Setting up efficient database access patterns for frontend applications

## Getting Started

This power contains comprehensive documentation and examples for using CloudBase Relational Database in web applications. After initialization, you can use familiar Supabase query patterns for all database operations.

## Important Notes

- Use synchronous initialization pattern for optimal performance
- Create a single shared database client and reuse across components
- After initialization, treat the database client exactly like a Supabase client
- Do not re-initialize CloudBase - share the client instance instead
